PICT Timetable System

Academic Timetable
Made Intelligent

Automatically generate conflict-free timetables for multiple divisions, shared faculty, and batch lab sessions — with real-time updates when anything changes.

🧠
CSP Algorithm
Backtracking solver with Most-Constrained Variable ordering. Zero teacher or room conflicts guaranteed across all divisions.
🔄
Real-Time Changes
Move slots, swap classes, handle teacher absences — automatic cascade resolution and faculty notifications.
🔔
Faculty Notifications
When a class is freed, all faculty get an alert. First to click "Claim" gets the slot — no double assignments.
🏷️
Batch Lab Scheduling
Each batch gets its own lab teacher and time. Labs auto-placed in any free lab room as 2 consecutive hours.
📅
Smart Room Allocation
Back-to-back lectures stay in the same classroom. Labs use any available lab. No fixed rooms needed per division.
⬇️
Excel Export
Download the full timetable as a formatted Excel workbook with all divisions, workload summary and change log.
